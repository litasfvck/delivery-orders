# Delivery Orders App

Минималистичное fullstack-приложение для оформления и просмотра заказов на доставку.

Приложение демонстрирует полный поток данных:

**PostgreSQL → .NET Web API → React UI**

Пользователь может создать новый заказ, просмотреть список всех заказов и открыть любой заказ в режиме чтения.

---

## Запуск

```bash
git clone https://github.com/litasfvck/delivery-orders
cd <repo>
docker compose up --build
```

После запуска:

* Frontend: http://localhost:4200
* Backend (Swagger): http://localhost:8080/swagger
* PostgreSQL: localhost:5432

---

## Функционал

### Создание заказа

Доступна форма создания заказа со следующими обязательными полями:

* Город отправителя
* Адрес отправителя
* Город получателя
* Адрес получателя
* Вес груза
* Дата забора груза

При создании заказа автоматически генерируется номер заказа.

### Список заказов

Все созданные заказы отображаются в отдельном списке.

Для каждого заказа отображаются:

* Номер заказа
* Город отправителя
* Адрес отправителя
* Город получателя
* Адрес получателя
* Вес груза
* Дата забора груза

### Просмотр заказа

При выборе заказа из списка открывается страница просмотра в режиме чтения с полной информацией о заказе.

## API Endpoints

### Получить список заказов

```http
GET /api/orders
```

Возвращает список всех созданных заказов.

### Получить заказ по идентификатору

```http
GET /api/orders/{id}
```

### Создать новый заказ

```http
POST /api/orders
```

Пример тела запроса:

```json
{
  "senderCity": "Moscow",
  "senderAddress": "Lenina 10",
  "receiverCity": "Kazan",
  "receiverAddress": "Baumana 5",
  "weight": 15.5,
  "pickupDate": "2026-07-17T00:00:00Z"
}
```
